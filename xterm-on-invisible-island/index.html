Lynx current development

We are currently working on Lynx 2.9.0.

Lynx's version numbers have "dev", "pre" or "rel" suffixes. For example, "lynx2.8.9dev.1" was a development version as part of the successive versions culminating in "lynx2.8.9rel.1".

This page contains links to the most recent Lynx development source code.

Language Support

Lynx supports multiple languages through use of gettext:

  • The translation project's page for Lynx summarizes the versions of the message catalogs which have been used.
  • Message catalogs from the translation project are bundled in the source distribution's po directory.
  • The About NLS page gives some historical information from when Lynx first started using these catalogs.

Questions or comments can be directed to the lynx-dev@nongnu.org mailing list,
which is archived at http://lists.nongnu.org/archive/html/lynx-dev/
